Tabs auto closer

Browser extension automatically closes invite tabs zoom whatsapp telegram and more after a timeout.

Such as:

  • https://company.zoom.us/j/*
  • https://t.me/joinchat/*
  • https://chat.whatsapp.com/*

And others at your discretion.

Example:

  • https:\/\/(.)*\.zoom.us\/j\/(.)* - regular expression for close Zoom
  • https:\/\/t\.me\/(.)* or https:\/\/telegram\.me\/(.)* - regular expression for close Telegram

Config for import:

{
        "version": "TabsAutoCloserExportV1",
        "data": {
            "groups": [
                {
                    "name": "whatsapp",
                    "matches": [
                    "https:\\/\\/chat\\.whatsapp\\.com\\/(.)*"
                    ],
                    "closeTimeout": 7000
                },
                {
                    "name": "Telegram",
                    "matches": [
                    "https:\\/\\/telegram\\.me\\/(.)*",
                    "https:\\/\\/t\\.me\\/(.)*",
                    "https:\\/\\/t\\.me\\/.*"
                    ],
                    "closeTimeout": 10000
                },
                {
                    "name": "zoom",
                    "matches": [
                    "https:\\/\\/(.)*\\.zoom.us\\/j\\/(.)*"
                    ],
                    "closeTimeout": 7000
                }
            ]
        }
}
